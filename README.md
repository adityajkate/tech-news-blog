### 🚀 Quick Start

Ensure you have Node.js 18+ and set up your `.env` variables (`GEMINI_API_KEY`, etc).

**1. Boot the Backend**
```bash
cd backend
npm install
npm run dev
```

**2. Boot the Frontend**
```bash
cd frontend
npm install
npm start
```

**3. Explore**
Navigate to `http://localhost:3000` to interact with your intelligent news feed.
